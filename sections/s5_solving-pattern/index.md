# Solving Pattern
## 1. Frequency
- example: `{a: 1, b: 2}`
## 2. Multiple pointers
- example: `left & right`
## 3. Sliding Window
- keep track of a subset of data in an `array` or `string`.
## 4. Divide and conquer
- divide a data into smaller chunks (sub-data) then apply a process for every chunk.
